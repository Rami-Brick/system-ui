# Fidelity Rubric

Use this before coding and again after the playground render.

## Fidelity Modes

### Close Clone

Preserve:

- layout proportions
- surfaces and colors
- typography scale
- spacing density
- imagery placement
- component shapes
- content structure

Only change content/assets when licensing or user instruction requires it.

### Inspired System

Preserve:

- major layout DNA
- signature components
- visual density
- surface atmosphere
- color relationships
- typography hierarchy
- role and scale of imagery

May adapt:

- names/copy/data
- exact icon choices
- component APIs
- brand naming
- asset sources

Do not use "inspired" as permission to create a generic redesign.

### Style Transfer

Preserve:

- mood
- palette direction
- typography direction
- shape language
- interaction style

May change:

- layout
- content structure
- component taxonomy

## Comparison Categories

Score each 0-2:

| Category | 0 | 1 | 2 |
|---|---|---|---|
| Layout | major regions changed | recognizable but proportions off | close region structure and proportions |
| Density | much too sparse/dense | somewhat off | close rhythm |
| Surfaces | wrong/flat | partially similar | close atmosphere |
| Type | wrong scale/personality | acceptable | close hierarchy/personality |
| Color | generic/wrong | approximate | close relationships |
| Signature Elements | missing | partial | clearly present |
| Assets | missing/collapsed | placeholders with weak sizing | preserved role and sizing |
| Supporting References | ignored | partially applied | palette/type/assets reflected in tokens |

If total is below 12 for close clone or below 11 for inspired system, iterate before final handoff.
